# Docling pulls in torch as a transitive dependency. To get GPU acceleration
# (5-20x faster extraction on CUDA), install a CUDA wheel BEFORE running pip
# install -r requirements.txt, e.g.:
#
#   pip install torch --index-url https://download.pytorch.org/whl/cu121
#
# Or for ROCm / MPS, see https://pytorch.org/get-started/locally/.
# The accelerator is auto-detected at startup (CUDA → MPS → CPU). Override
# with DOCLING_DEVICE=cpu|cuda|mps. Disable OCR (default) with DOCLING_DO_OCR=0;
# enable for scanned docs with DOCLING_DO_OCR=1. Tune CPU threads with
# DOCLING_NUM_THREADS=<N>.
