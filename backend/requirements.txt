# Docling pulls in torch as a transitive dependency. To get GPU acceleration
# (5-20x faster extraction on CUDA), install a CUDA wheel BEFORE running pip
# install -r requirements.txt, e.g.:
#
#   pip install torch --index-url https://download.pytorch.org/whl/cu121
#
# Or for ROCm / MPS, see https://pytorch.org/get-started/locally/.
# The accelerator is auto-detected at startup (CUDA → MPS → CPU). Override
# with DOCLING_DEVICE=cpu|cuda|mps. OCR is decided per-document by sampling
# extractable text via pypdfium2: digital PDFs run on the fast no-OCR
# pipeline, image-only PDFs route through the OCR converter automatically.
# Force the decision globally with DOCLING_DO_OCR=0 (always off) or
# DOCLING_DO_OCR=1 (always on). Tune CPU threads with DOCLING_NUM_THREADS=<N>.
